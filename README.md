# E-Shopping Website : HTML | CSS | React.js | Context API

A responsive and interactive e-commerce web application built using React.js, focused on component-based architecture, global state management, and dynamic cart functionality.
This project demonstrates practical frontend development skills including reusable UI components, React hooks, and application-level state handling.


# Project Overview :

1.The E-Shopping Website enables users to browse products, manage cart items, and simulate authentication in a clean, responsive interface.
2.The application is designed using a modular React architecture and leverages React Hooks for efficient state handling.

# Core Features :

1.User Authentication (Login/Logout UI flow)
2.Product Listing with dynamic rendering
3.Add to Cart / Remove from Cart functionality
4.Cart quantity management
5.Real-time UI updates
6.Fully responsive design

# Technical Highlights
1.Component-Based Architecture: Built using reusable and structured React components to ensure maintainability and scalability.

2.Global State Management:  Implemented useState and useContext for centralized cart management and state sharing across components.

3.React Hooks:
Utilized:
useState for managing component state
useContext for global state handling
Functional components with modern React practices

4.Dynamic Rendering: Products and cart items are rendered dynamically using JavaScript array methods like map().

5.Responsive UI: Designed with clean CSS structure ensuring cross-device compatibility.            

# Project Objective :
The goal of this project was to build a scalable frontend e-commerce application while applying:
1.React Hooks
2.Context API
3.Component reusability principles
4.Real-time state synchronization
5.Clean UI/UX design practices
